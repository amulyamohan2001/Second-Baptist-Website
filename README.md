# Second Baptist Website Redesign

## Project Overview

Redesigned and modernized an the organizations website originally built as a single-page Wix site. Reorganized content into a professional multi-page website, improved navigation and user experience, updated outdated information, and established a stronger digital presence for the church community.

## Responsibilities

* Redesigned the website architecture and navigation structure
* Created a modern landing page to improve visitor engagement
* Migrated and organized content across dedicated website pages
* Removed outdated announcements and refreshed website content
* Developed online forms to streamline communication and visitor inquiries
* Integrated Kit email marketing for newsletter subscriptions and outreach campaigns
* Applied JavaScript and web development knowledge to enhance website functionality
* Optimized the website for desktop, tablet, and mobile devices
* Managed website content and design using Hostinger Website Builder

## Website Purpose

The redesigned website serves as a central hub for church information, ministry updates, events, visitor engagement, and digital outreach. The project focused on improving accessibility, usability, and communication for both members and first-time visitors.
